# Real-World Data Sources and References

The synthetic data generated in this project is informed by patterns observed in
publicly available and credible real-world sources.

## Referenced Sources

- **Asana Public Project Templates**
  Used to understand common project structures, section names
  (e.g., Backlog, To Do, In Progress, Done), and task organization patterns.

- **Asana "Anatomy of Work" Reports**
  Referenced for task completion rates, collaboration patterns,
  and enterprise productivity benchmarks.

- **GitHub Issue Trackers**
  Used to derive realistic task naming conventions, description styles,
  and lifecycle patterns commonly seen in engineering workflows.

- **ProductHunt Launch Pages**
  Referenced for project and campaign naming patterns in SaaS companies.

- **Y Combinator Company Directory**
  Used to understand common SaaS company naming conventions and organizational scale.

## Notes on Data Usage

No live scraping or automated data collection was performed.
All data is synthetically generated, and real-world sources were used only
to derive high-level patterns in compliance with terms of service.
